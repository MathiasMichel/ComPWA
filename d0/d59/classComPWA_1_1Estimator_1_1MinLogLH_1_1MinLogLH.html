double MinLogLH::calcPenalty ( )
virtual

Calculate penalty term A penalty term can be added to the LH to punish unnecessary complexity of amplitudes in minimization. The strength of this term can be set via MinLogLH::setPenaltyScale(double). A detailed description of the approach can be found in arXiv:1505.05133
